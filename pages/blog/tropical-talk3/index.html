Reminder

Fix an algebraically closed field kkk. An nnn-pointed nodal curve (C,P1,...,Pn)(C, P_1,...,P_n)(C,P1​,...,Pn​) of genus ggg over kkk is a projective curve with arithmetic genus ggg and at most nodal singularities. The curve is stable if it is connected and the automorphism group Aut⁡(C,P1,...,Pn)\operatorname{Aut}(C, P_1,...,P_n)Aut(C,P1​,...,Pn​) is finite. The space M‾g,n\overline{\mathcal M}_{g,n}Mg,n​ is the moduli space of all nnn-pointed curves of genus ggg. Note that this is a Deligne-Mumford stack because all nnn-pointed curves have finitely many automorphisms.

Definition 1: .   For each nnn-pointed curve with at most nodes as singularities over k‾\overline{k}k, we can assign it a weighted dual graph GC=(V,E,L,h)G_C = (V,E,L,h)GC​=(V,E,L,h) given by

  1. the set of verticies V=V(G)V = V(G)V=V(G) is the set of irreducible components of CCC and

  2. the set of edges E=E(G)E = E(G)E=E(G) is the set of nodes of CCC where an edge e∈Ee \in Ee∈E is incident to verticies v1,.v2v_1,.v_2v1​,.v2​ if the corresponding node lies in the intersection of the corresponding components. Note that a node contained in only one component corresponds to a loop.

  3. The ordered set of legs L=L(G)L = L(G)L=L(G) correponds to the marked points. These are sometimes called "half edges", they're homeomorphic to an unbounded ray in R\mathbb RR.

  4. the function h:V→Nh:V\to \mathbb Nh:V→N is the genus function, h(v)h(v)h(v) is the geometric genus of the component v∈Vv\in Vv∈V. This produces a weighted unbounded graph.

Tropical curves and their moduli
